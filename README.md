# Sistema de Reserva de Hotel (Java)

Aplicação em Java que simula o gerenciamento de reservas de hotel, permitindo cadastrar e atualizar datas de hospedagem e calcular a duração da estadia.

## Funcionalidades
- Cadastro de reserva (check-in e check-out)
- Cálculo automático da duração da estadia
- Atualização das datas da reserva
- Validação de datas

## Conceitos aplicados
- Programação Orientada a Objetos (POO)
- Manipulação de datas
- Tratamento de exceções
- Regras de negócio
- Entrada de dados via terminal

## Tecnologias utilizadas
- Java
